# 🎯 Shooting Range (FPS Prototype)

## 📖 Overview
This project is a First-Person Shooter (FPS) shooting range simulator built in Unity. Players can practice their aim, reflexes, and weapon handling using a classic AK-47 assault rifle. The project features both a relaxed practice environment and a competitive time-trial challenge.

## 🎮 Game Modes

* **Free Mode:** A sandbox experience where points are awarded for every target hit. With no time pressure, it serves as the perfect environment to test weapon mechanics, practice recoil control, and improve general accuracy.
* **Challenge Mode:** A competitive time-trial mode where the objective is to shoot down 50 targets as quickly as possible. Designed to test both precision and speed, encouraging players to beat their high scores.

## ⚙️ Core Mechanics & Features

* **Raycast Shooting System:** Fast, instantaneous hit detection implemented using Unity's `Physics.Raycast` for precise shooting mechanics.
* **Interactive "Shoot-to-Interact" UI:** Physical in-game buttons that the player must shoot to trigger events, such as toggling the Challenge Mode or activating Infinite Ammo.
* **Dynamic Target Respawning:** Destroyed targets automatically respawn after a brief delay at randomized positions along their designated tracks, keeping the gameplay dynamic.
* **Weapon & Ammo Management:** Fully integrated ammo tracking UI and weapon reloading mechanics.
